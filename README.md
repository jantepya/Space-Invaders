# Space Invaders
------
## About
![Imgur](https://i.imgur.com/olLZbSF.png)

This is just a Javascript implementation of a Space Invaders type game I made for fun.

## Controls:

| Key | Action |
|-----|--------|
| Arrow Keys | Move left and right |
| Left Mouse Button| Shoot|
| R | Restart Game |
| P | Pause Game |
